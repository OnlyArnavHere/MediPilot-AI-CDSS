# MediPilot AI-CDSS

AI-Based Clinical Decision Support System (AI-CDSS) web app.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Local AI via Ollama (`http://localhost:11434/api/generate`)

## Run Locally

1. Install dependencies:
   - `npm install`
2. Pull model:
   - `ollama pull llama3`
3. Start app:
   - `npm run dev`
